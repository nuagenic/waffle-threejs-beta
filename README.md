# Run Server

- 루트 디렉토리에서 아래 명령어로 실행 가능합니다.

  - `bottega-veneta`: `npm run bottega`
  - `waffle-room`: `npm run waffleroom`

- 별도로 설치할 라이브러리가 많지 않을 것으로 예상되어 우선은 별도의 vite config를 두는 것으로 조치했으며, 차후 workspace로 관리할 수도 있습니다.
